* Helm, an Incremental Completion Framework
There are many helm things. I use it a lot.

A lot of things are taken from taken from
https://tuhdo.github.io/helm-intro.html

Before we load any helm things, need to load [[https://github.com/PythonNut/helm-flx][helm-flx]] so it uses flx instead of
helm's fuzzy matching.

#+BEGIN_SRC emacs-lisp
(use-package helm-flx
 :ensure t
 :init (helm-flx-mode +1))
#+END_SRC

* Multi-compile for different compilations I run

So, there are a bunch of ways to compile things in the ES codebase, so I'm
checking out [[https://github.com/ReanGD/emacs-multi-compile][multi-compile]] to see if it works well for this sort of thing.

#+BEGIN_SRC emacs-lisp
(use-package multi-compile
  :ensure t
  :commands multi-compile-run
  :bind ("<f5>" . multi-compile-run)
  :config
  (require 's)
  (setq multi-compile-completion-system 'helm
        multi-compile-alist
        '(;; Elasticsearch-specific compilation commands
          ((s-contains? "es/elasticsearch" default-directory)
           . (("ES core compile" .
               "gradle :core:precommit :core:compileJava :core:compileTestJava --console=plain")
              ("ES everything compile" .
               "gradle precommit compileJava compileTestJava --console=plain")
              ("ES precommit" .
               "gradle precommit --console=plain")))
          ;; X-pack specific compilation commands
          ((s-contains? "es/x-plugins" default-directory)
           . (("x-pack compile" .
               "gradle :x-plugins:elasticsearch:x-pack:compileJava :x-plugins:elasticsearch:x-pack:compileTestJava --console=plain")
              ("x-pack everything compile" .
               "gradle precommit compileJava compileTestJava --console=plain")
              ("x-pack precommit" .
               "gradle precommit --console=plain")))
          ;; Java compilation commands
          (java-mode
           . (("ant compile" .
               "ant compile compile-test")
              ("mvn compile" .
               "mvn compile test-compile")
              ("gradle compile" .
               "gradle compileJava compileTestJava --console=plain")))))
  ;; Make M-x compile invoke multi-compile
  (defalias 'compile 'multi-compile-run))
#+END_SRC
